# Geowi Docs
## Geowi Docsとは？
オープンソースかつマークダウン方式のwiki

## 今回なぜ利用しようと思ったか？
少人数の会社で社内手順ほぼなく(特に人事関連や営業関連のフロー等)  
自由にそれぞれが判断して対応している部分が多い点が課題だと感じた  
↓  
社内wikiを利用したいけど、あまりお金をかけられない  
検索していたところ、オープンソースかつ比較的使いやすそうなUI!!  
サーバーに少し空きもある！  
↓  
使ってみよう(^^)/   
 
 
 
## 対応OS
- docker-compose
- Ubuntu Server
- CentOS  


## どこに構築するか？
対応しているOSが結局対応していないため、仮想でどこかにサーバー建てられないか検討
